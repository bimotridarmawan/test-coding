Detail Modul

Modul 1: Pengenalan HTML & CSS

Pada modul pertama ini, Anda akan belajar tentang dasar-dasar HTML dan CSS, dua teknologi utama yang digunakan untuk membuat struktur dan tampilan website. Modul ini mencakup pembuatan halaman web statis menggunakan elemen HTML dan penerapan gaya dengan CSS.

  • Pengenalan HTML: Elemen, tag, dan struktur dasar HTML.
  • CSS Dasar: Selektor, properti, dan cara menata elemen.
  • Layout dan desain dasar dengan CSS.

Modul 2: JavaScript Dasar

Modul ini memperkenalkan JavaScript, bahasa pemrograman yang memungkinkan Anda untuk menambahkan interaktivitas pada website. Anda akan belajar tentang variabel, tipe data, fungsi, dan kontrol alur seperti perulangan dan pengkondisian.

  • Pengantar JavaScript: Variabel, tipe data, dan operator.
  • Kontrol Alur: Pengkondisian dan perulangan.
  • Fungsi dan event handling pada JavaScript.

Modul 3: Responsive Design

Modul ini mengajarkan bagaimana membuat website yang responsif, sehingga tampilannya dapat menyesuaikan dengan ukuran layar perangkat yang digunakan. Anda akan belajar teknik seperti media queries dan flexible grid layouts.

  • Pengenalan desain responsif dan pentingnya pengalaman pengguna.
  • Penggunaan media queries untuk menyesuaikan tampilan.
  • Penggunaan Flexbox dan Grid untuk layout responsif.

Modul 4: Framework Modern (React & Vue)

Pada modul terakhir ini, Anda akan dikenalkan dengan dua framework JavaScript populer: React dan Vue. Anda akan belajar bagaimana membangun aplikasi interaktif menggunakan komponen dan state management.

  • Pengantar React: Komponen, props, dan state.
  • Pengantar Vue: Menggunakan Vue untuk membangun aplikasi sederhana.
  • Perbandingan antara React dan Vue: Keunggulan dan kelemahan masing-masing framework.
