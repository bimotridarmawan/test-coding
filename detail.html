Pengembangan Web

Pengajar: Bimo Tri Darmawan

Pelajari cara membuat website responsif dengan HTML, CSS, dan JavaScript. Anda juga akan diperkenalkan dengan framework modern seperti React dan Vue.

  • Durasi Kursus: 4 Minggu
  • Metode Pembelajaran: Video tutorial, latihan praktek, dan kuis.
  • Persyaratan Awal: Tidak ada pengalaman coding yang dibutuhkan. Dasar-dasar HTML dan CSS akan sangat membantu.
  • Sertifikat: Sertifikat akan diberikan setelah kursus selesai.
  • Harga Kursus: Rp 500.000 (Diskon 20% untuk pendaftaran awal!)
